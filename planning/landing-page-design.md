# Landing Page Design

The public-facing marketing page for SurveyGo, following the UX philosophy of clean, minimal design inspired by Notion and Linear.

---

## Design Philosophy

- **Clean & Minimal**: Notion-inspired simplicity
- **Action-First**: Clear CTAs above the fold
- **Trust Building**: Social proof prominently displayed
- **Performance**: Fast loading, optimized images
- **Mobile-First**: Responsive from the start

## Animations & Interactions

- **Hero image**: Subtle floating animation
- **Feature cards**: Fade-in on scroll
- **CTAs**: Hover scale (1.02) + shadow
- **Testimonials**: Auto-carousel with manual control
- **Numbers**: Count-up animation when in view
- **Mobile menu**: Slide-in from right

## Performance Targets

- **Lighthouse Score**: > 95
- **First Contentful Paint**: < 1.5s
- **Time to Interactive**: < 2.5s
- **Core Web Vitals**: All green
- **Image optimization**: WebP, lazy loading
- **Bundle size**: < 200KB initial JS

---

## A/B Test Ideas (Post-Launch)

1. Hero headline variations
2. CTA button text ("Start Free" vs "Get Started" vs "Create Survey")
3. Social proof placement (above vs below fold)
4. Pricing visibility (show on landing vs separate page)
5. Demo video vs static screenshot
